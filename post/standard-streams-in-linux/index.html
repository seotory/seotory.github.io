лҰ¬лҲ…мҠӨм—җм„ң std(standard streams)лһҖ

2016-04-16

лҰ¬лҲ…мҠӨм—җм„ң н”„лЎңк·ёлһЁмқҙ кө¬лҸҷмӢң н”„лЎңк·ёлһЁкіј мӢӨн–үнҷҳкІҪ мӮ¬мқҙлҘј мҶҢнҶөн•ҳкё° мң„н•ҙм„ңлҠ” лҜёлҰ¬ м—°кІ°лҗң data streamмқҙ н•„мҡ”н•ҳлӢӨ. н”„лЎңк·ёлһЁм—җ кё°ліём ҒмңјлЎң н‘ңмӨҖмһ…л Ҙ, н‘ңмӨҖм¶ңл Ҙ, н‘ңмӨҖм—җлҹ¬ 3к°Җм§Җмқҳ лҚ°мқҙн„° мҠӨнҠёлҰј(data stream)мқҙ м—°кІ°(connected)лҗңлӢӨ. мқҙ 3к°Җм§Җмқҳ I/O м»Өл„Ҙм…ҳмқ„ мҠӨн…җлӢӨл“ң мҠӨнҠёлҰј(standard streams) мҰү STDлқјкі  л§җн•ңлӢӨ.

мӣҗлһҳ I/O лҠ” л¬јлҰ¬м ҒмқҖ мһҘм№ҳм—җ мқҳн•ҙ л°ңмғқн•ҳм§Җл§Ң, STDм—җм„ң мқҙм•јкё°н•ҳлҠ” I/OлҠ” мқҙкІғліҙлӢӨ мўҖ лҚ” м¶”мғҒм ҒмқҙлӢӨ. мқјл°ҳм ҒмңјлЎң shellм—җм„ң лӘ…л №м–ҙк°Җ мӢӨн–үлҗ  л•Ң standard streamк°Җ м—°кІ°мқҙ лҗҳкі  pipeлҘј нҶөн•ҙ standard streamмқҖ м„ңлЎң м—°кІ°мқҙ лҗҳкё°лҸ„ н•ңлӢӨ. лҳҗн•ң л¶ҖлӘЁ н”„лЎңм„ёмҠӨм—җм„ң мһҗмӢқ н”„лЎңм„ёмҠӨлЎң standard streamмқҙ мғҒмҶҚлҗңлӢӨ.

3к°Җм§Җмқҳ лҚ°мқҙн„° мҠӨнҠёлҰјм—җлҠ” кі мң  мҲ«мһҗк°Җ л¶ҷм—¬м§Җл©° мқҙ лІҲнҳёлҠ” ліҖн•ҳм§Җ м•ҠлҠ”лӢӨ. STDлһҖ standardмқҳ м•ҪмһҗмқҙлӢӨ.
н‘ңмӨҖмһ…л Ҙ(STDIN) stream number 0
н‘ңмӨҖм¶ңл Ҙ(STDOUT) stream number 1
н‘ңмӨҖм—җлҹ¬(STDERR) stream number 2

STDINлҠ” нӮӨліҙл“ңлҘј нҶөн•ҙ мқјм–ҙлӮҳкі  лҸҷмһ‘ кІ°кіјлҠ” лӘЁлӢҲн„°м—җ STDOUTлҳҗлҠ” STDERRлЎң м¶ңл Ҙн•ңлӢӨ.

н‘ңмӨҖмһ…л Ҙ(STDIN)

STDINмқҙлһҖ н”„лЎңк·ёлһЁ м•ҲмңјлЎң stream dataк°Җ л“Өм–ҙк°ҖлҠ” кІғмқҙлӢӨ. лі„лҸ„мқҳ лҰ¬лӢӨмқҙл үм…ҳмқҙ м—Ҷмңјл©ҙ нӮӨліҙл“ңлҘј нҶөн•ҙ STDINмқ„ л°ӣкІҢлҗңлӢӨ.

$ ls -al

н‘ңмӨҖм¶ңл Ҙ(STDOUT)

н‘ңмӨҖ м¶ңл ҘмқҙлһҖ н”„лЎңк·ёлһЁмқ„ мӢӨн–үмӢңмј°мқ„ кІҪмҡ° н„°лҜёл„җлЎң лӮҳмҳӨлҠ” кІ°кіј(data)лҘј л§җн•ңлӢӨ.

н‘ңмӨҖм—җлҹ¬(STDERR)

н‘ңмӨҖм—җлҹ¬лһҖ н”„лЎңк·ёлһЁмқҙ мӢӨн–үмӢңм—җ л°ңмғқн•ҳлҠ” м—җлҹ¬м—җ н•ҙлӢ№н•ңлӢӨ.

$ ls -al test.txt test.test
ls: test.test: No such file or directory
-rw-r--r--  1 zion437  staff  53  4 16 21:00 test.txt

2лІҲм§ё лқјмқём—җм„ң test.test нҢҢмқјмқ„ м°ҫм§Җ лӘ»н•ҳм—¬ н‘ңмӨҖм—җлҹ¬к°Җ л°ңмғқн•ҳмҳҖлӢӨ.

лҰ¬лӢӨмқҙл үм…ҳмқ„ мқҙмҡ©н•ҙ н‘ңмӨҖмһ…л Ҙ, н‘ңмӨҖм¶ңл Ҙ, н‘ңмӨҖм—җлҹ¬ мӮ¬мҡ©н•ҙліҙкё°

мһҗ мқҙм ң мң„мқҳ лӮҙмҡ©мқ„ нҶ лҢҖлЎң лҚ°мқҙн„° мҠӨнҠёлҰјмқ„ нҢҢмқјлЎң кё°лЎқмқ„ н•ҙліҙмһҗ. мЈјлЎң >, >>, < м—°мӮ°мһҗлҘј мӮ¬мҡ©н•ҳкІҢ лҗ  кІғмқҙлӢӨ.

лҰ¬лӢӨмқҙл үм…ҳ н‘ңмӨҖмһ…л Ҙ

н‘ңмӨҖмһ…л Ҙмқ„ м»Өл§Ёл“ңм°Ҫм—җм„ң мқҙмҡ©н•ҳл Өл©ҙ < м—°мӮ°мһҗлҘј нҷңмҡ©н•ңлӢӨ. wcлҠ” лқјмқёмқҳ мҲҳлҘј м№ҙмҡҙнҠён•ҳлҠ” мӢӨн–ү лӘ…л №м–ҙмқёлҚ°, мқҙ лӘ…л №м–ҙлҘј нҷңмҡ©н•ҙм„ң мң„м—җм„ң л§Ңл“Өм—ҲлҚҳ test.txt нҢҢмқјмқ„ н‘ңмӨҖмһ…л ҘмңјлЎң м „лӢ¬н•ҙліҙмһҗ.

  1. н‘ңмӨҖмһ…л Ҙ нҷңмҡ©

    $ wc < test.txt 
      20     173    1094

  2. н‘ңмӨҖмһ…л Ҙ лҜёнҷңмҡ©

    $ wc test.txt 
      20     173    1094 test.txt

1лІҲкіј 2лІҲмқҳ кІ°кіјлҘј ліҙл©ҙ нҷ•мқё н•  мҲҳ мһҲл“Ҝмқҙ < м—°мӮ°мһҗлҘј л„Јм§Җ м•Ҡм•„лҸ„ кІ°кіјк°Җ лҳ‘к°ҷмқҢмқ„ м•Ң мҲҳ мһҲлӢӨ. кІ°кіјк°Җ лҳ‘к°ҷлӢӨкі  н•ҙм„ң wc мқҳ мһ‘лҸҷ л©”м»ӨлӢҲмҰҳмқҙ лҳ‘к°ҷмқҖ кІғмқҖ м•„лӢҲлӢӨ. 1лІҲ к°ҷмқҖ кІҪмҡ°м—җлҠ” wc н”„лЎңк·ёлһЁмқҙ мӢӨн–үлҗҳл©ҙм„ң м—°кІ°лҗҳм—ҲлҚҳ 3к°ңмқҳ мҠӨнҠёлҰј(н‘ңмӨҖмһ…л Ҙ, м¶ңл Ҙ, м—җлҹ¬) мӨ‘м—җм„ң н‘ңмӨҖмһ…л Ҙмқҳ мҠӨнҠёлҰјмқҙ мқҙмҡ©лҗҳм–ҙ test.txtк°Җ м „мҶЎлҗңлӢӨ. 2лІҲмқҖ wc н”„лЎңк·ёлһЁ лӮҙл¶Җм—җм„ң test.txt нҢҢмқјмқ„ мқҪм–ҙл“Өм—¬ мӢӨн–үн•ҳкІҢ лҗңлӢӨ.

лҰ¬лӢӨмқҙл үм…ҳ н‘ңмӨҖм¶ңл Ҙ

лЁјм Җ н‘ңмӨҖм—җлҹ¬к°Җ м—ҶлҠ” мӢӨн–үл¬ём—җ лҢҖн•ҙ нҢҢмқјлЎң л§Ңл“Өм–ҙ ліј кІғмқҙлӢӨ.

ls -al > test.txt

мң„мқҳ лӘ…л №м–ҙлҘј мӢӨн–үн•ҳл©ҙ н„°лҜёл„җ м°Ҫм—җм„ң лҚ°мқҙн„°к°Җ лӮҳмҳӨм§Җ м•ҠлҠ”лӢӨ. мқҙмң лҠ” лҚ°мқҙн„° мҠӨнҠёлҰјмқҙ л°”лЎң test.txtлЎң мқҙлҸҷлҗҳм–ҙ нҢҢмқјлЎң мһ‘м„ұлҗҳм—Ҳкё° л•Ңл¬ёмқҙлӢӨ. test.txt нҢҢмқјмқ„ м—ҙм–ҙліҙмһҗ.

мЈјмқҳн•  м җмқҖ > мӢӨн–үмӢң к°ҷмқҖ нҢҢмқјлЎң м§Җм •н•ҳл©ҙ лҚ°мқҙн„°к°Җ лӢӨмӢң м“°мқҙлҠ”кІҢ м•„лӢҲлқј лӘЁл‘җ м§ҖмӣҢм§ҖкІҢ лҗңлӢӨ. > лӘ…л №м–ҙк°Җ мӢңмһ‘ м „м—җ лҢҖмғҒнҢҢмқјмқ„ 0byteлЎң л§Ңл“Өкі  мӢңмһ‘н•ҳкё° л•Ңл¬ёмқҙлӢӨ.

лҰ¬лӢӨмқҙл үм…ҳ н‘ңмӨҖм—җлҹ¬

н„°лҜёл„җ м°Ҫм—җм„ң м•„лһҳмҷҖ к°ҷмқҙ мһ…л Ҙн•ҙліҙмһҗ.

ls: test.test: No such file or directory

мҰү н‘ңмӨҖм—җлҹ¬лҘј нҢҢмқјлЎң лӢҙкё° мң„н•ҙм„ңлҠ” н‘ңмӨҖм—җлҹ¬ stream number 2лҘј л¶ҷм—¬м„ң м•„лһҳмҷҖ к°ҷмқҙ нҳём¶ңн•ҙм•ј н•ңлӢӨ.
